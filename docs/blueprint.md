# **App Name**: RoofForce OS

## Core Features:

- Proactive Relationship Building: Enable salespeople to log and track interactions (in-person, phone, email) with property managers, facilities managers, and building owners.
- Lead Generation & Opportunity Tracking: Systematically track leads, opportunities, and associated data like property details and customer LTV.
- Account & Property Management: Centralized management of accounts, contacts, properties, and roles, linking all relevant data points.
- KPI Tracking & Goal Setting: Visual dashboards and reporting to track key performance indicators (KPIs) and progress towards goals.
- Automated Task Generation: Use AI tool to create a series of tasks or reminders to ensure timely follow up and nurturing of prospects
- Simple/one tap activity logging: Simple/one tap activity logging, account creation, property creation, contact creation, etc. for field use
- Prospect Web Search: Integrate web search to find new prospects and add them directly to the lead tracking system.
- Secure Login: Implement a secure login system for user authentication.
- Backend Lead Tracking: Develop a backend system for tracking leads and managing customer data.
- Map Feature for Leads: Implement a map feature to find "ugly roofs" and add them as leads.

## Style Guidelines:

- Primary color: Deep sky blue (#38b6ff), suggesting reliability and innovation.
- Background color: Very light blue (#E5F6FF), to create a clean, professional interface.
- Accent color: Blue-violet (#833ab4), for highlights and calls to action, adding a touch of modern sophistication.
- Body and headline font: 'Inter', a grotesque sans-serif for a modern and neutral look.
- Use clear, professional icons for navigation and data visualization.
- Clean, data-focused layout emphasizing key metrics and customer information.
- Subtle transitions and animations to enhance user experience during navigation and data loading.
